# 📸 Responsive Image Gallery

A modern and responsive **Image Gallery Web Application** built using **HTML5, CSS3, and JavaScript** as part of the **CodeAlpha Frontend Development Internship**.

The project provides an interactive and visually appealing gallery where users can browse images, filter them by category, and view them in an elegant lightbox with navigation controls.

## 🎯 Functionalities

### 🏠 Home Page
- Displays a responsive image gallery.
- Shows category filter buttons.

### 🎨 Category Filters
Users can filter images by:

- All
- Nature
- City
- Food
- Technology

### 📸 Lightbox
Clicking any image opens it in a larger preview.

The lightbox supports:

- Previous Image
- Next Image
- Close Button

### 📱 Responsive Design
The gallery automatically adjusts based on the screen size.

- Desktop
- Laptop
- Tablet
- Mobile
